Loan Creation Guide

Welcome, Admin! To create a new loan, follow these steps:

  1. Enter the name of the loan in the "Loan name" field.
  2. Specify the maximum loan amount in the "Maximum amount" field.
  3. Select the desired payment options by checking the corresponding checkboxes.
  4. Enter the interest percentage in the "Interest percentage %" field (e.g., 0.3 for 30%).
  5. Click the "Create" button to generate the new loan.

That's it! You've successfully created a new loan. If you have any questions, feel free to contact support.
